# TrabalhoAppContatos
Trabalho em equipe da disciplina de Desenvolvimento Mobile na Universdade Positivo. O projeto em destaque é sobre um aplicativo funcional de gerenciamento de contatos. Os integrantes do grupo são Lincoln Gabriel Sousa Matos, Arthur João Padilha Conci, Heitor Augusto Andrade e Thiago Amaral Costa.
## Como funciona?
Para utilizar o app, o usuário deve se cadastrar com o nome de usuário, email e senha logo no início quando abre a aplicação. Há uma aba de digitar o número de telefone, uma aba para gerar QR code e uma aba da lista de contatos.
